# Portfolio Website

## Clone the Repository

```bash
git clone https://github.com/AnudeepthiKolagani/anudeepthi-workspace.git
cd anudeepthi-workspace
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
